Workflow: Apps and functionality

Currently, can invoke:

  • Productivity/General: Wunderlist, Evernote, Skype, Uber, Editorial, Pythonista
  • Device: Location, network, brightness, flashlight, volume, notifications, music playback control,
  • Sharing: Clipboard, message/emails, FB, Twitter, Tweetbot 3, Airdrop, Instagram, Slack, Tumbler, share extension

Workflow: Scripting

get/set/name/concat vars, program flow (if, repeat, wait)

'stdlib': Basic math, b64 enc, get hash, some regex, basic input

also: "Content graph" conversion system

Pythonista

In short: Python env on iOS

Including: syntax highlighting in editor, auto-completion, REPL terminal

Standard library, and then some

A number of modules specifically for Pythonista on iOS

Pythonista: General

Benefits of a 'normal' programming language (relative to iOS)

Classic functionality: things like running a server, small app

Data persistence: You can create a sqlite DB! (among other py modules)

Pythonista: iOS Specific

Can invoke via Workflow, also directly from share extension, grab image/text/URLs

Access to services/data: location, motion, photos, reminders, notifications, bluetooth peripherals, contacts, text-to-speech, and more

Eek out more(?): Obj-C bridge
